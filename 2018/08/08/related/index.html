吉迈机器人资料汇总

布科思技术支持资料网站

机器人流程化配置方法（分步）

步骤1. 配置网络

布科思的主机出厂配置默认连接的WIFI网络是：

网络名称：robot_test

网络密码：robot123

在有这样设置的WIFI无线路由器的情况下会自动连接。如果需要设置成自动连接其他的WIFI网络，可以通过两种方法：

  1. 通过ssh远程登录布科思主机修改，注意必须确保和布科思主机在同一个WIFI网络下。ssh命令

    ssh boocax@192.168.1.144
    'boocax@192.168.1.144's password:

    替换划掉的部分为实际的局域网内布科思主机地址。ssh远程登录密码为robot123，这也是布科思主机的root密码。
    登录布科思主机后可以修改位于主目录下 data 目录的 wlan.conf 文件

    boocax@boocax-robot:~$ sudo nano data/wlan.conf

    把 ssid 和 password 分别改为其他的网络名称和WIFI密码

    ssid=(网路名称)
    password=(WIFI密码)

    设置完WIFI后重启布科思就会自动连接新网络

  2. 用VGA视频传输线连接主机VGA输出到电脑屏幕。修改 wlan.conf 的方法同上。
